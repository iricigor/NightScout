# Nightscout API wrapper

Lear more about Nightscout [here](https://nightscout.github.io/)

## Quick start

```ps
Import-Module ./Nightscout.psm1
Set-BGServer https://myserver.domain:port
Get-BGLastValue -Verbose
```
